This page show cases the research and development work done so far. The new widgets/components show cased here use following techniques.

  • Only core dojo is used for most of the widgets. Dijit is used sparsely for widget life cycle.
  • Templates are extensively used to speed up the dom generation.
  • less css framework is used to externalize the style declarations.
  • No images will used for anything after everything is migrated to lesscss.
